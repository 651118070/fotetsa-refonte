Développement itératif, évolutif & Agile

Chacune de nos applications web est conçue, pensée, architecturée et développée comme un produit unique. Nos équipes d’ingénieurs sont impliquées dès l’avant vente, puis dans la phase de vision des projets.

S’appuyant sur la richesse de l’écosystème open-source, nos équipes vous recommandent les meilleures solutions en vous partageant leurs avantages et inconvénients. Vous devenez ainsi acteur des choix technologiques qui accompagneront votre entreprise pour plusieurs années.

Découvrez les meilleures technologies open-source pour répondre à vos besoins spécifiques.

Langages Programmation

JavaScript

JavaScript est un langage de script léger, orienté objet, principalement connu comme le langage de script des pages web.

TypeScript

TypeScript est une extension de Javascript qui permet de gérer et valider les types des données au sein de Javascript.

PHP

PHP est un langage open source, conçu pour le développement d’applications web.

Python

Python est un langage open source, gratuit qui est porté par une forte communauté.

Rust

Rust est un langage multi-paradigme conçu et développé par Mozilla.

Frameworks Front-end

React

React est une bibliothèque JavaScript open source développée par Facebook pour faciliter la création d’applications web.

Angular

Angular est un framework open source écrit en JavaScript qui permet la création d’applications web.

Vue.js

Vue.js est un framework JavaScript open-source pour construire des interfaces utilisateur et des applications web monopages.

Nuxt.js

Nuxt.js est un framework Vue open-source visant à améliorer l’expérience développeur (DX), lors des phases de développement d’un projet.

Frameworks Back-end

Node.js

Node.js est un environnement d’exécution JavaScript côté serveur.

NestJS

NestJS est un framework permettant de créer des applications Node.js

Symfony

Symfony est des frameworks PHP les plus utilisés pour développer des applications web.

Flask

Flask est un micro-framework web écrit en Python. Son objectif est de garder un noyau simple mais extensible.

Strapi

Strapi est un cms headless open-source basé sur Node.js et ReactJs.

Frameworks Mobile

React Native

React Native permet de concevoir rapidement des applications multi-plateformes au rendu natif, fluides et performantes.

Flutter

Flutter est le framework développé et soutenu par Google qui offre une alternative à React Native.

PWA

Une PWA est un site web ayant intégré le meilleur des fonctionnalités des applications et des navigateurs pour offrir une expérience native aux utilisateurs.

Outils DevOps

Kubernetes

Kubernetes est une solution open source permettant de gérer, déployer, et passer à l’échelle des applications containerisées.

Terraform

Terraform est une solution open source développée par la société Hashicorp qui permet de gérer son infrastructure cloud avec du code

Docker

Docker est un logiciel libre permettant de lancer des applications dans des conteneurs logiciels.

Ansible

Ansible est une plate-forme logicielle libre pour la configuration et la gestion des ordinateurs.

Outils No-Code

No-Code

No-Code : programmation sans saisir une ligne de code grâce à des outils graphiques.

Bubble

Bubble est une plateforme No-Code qui permet de créer facilement un site web ou une application mobile.

Airtable

Airtable est un outil Low-code qui permet de créer facilement, à partest un logiciel libre permettant de lancer des applications dans des conteneurs logiciels.ir de modèles, des bases de données prêtes à l’emploi.

Make

Make (anciennement Integromat) est une plateforme d’automatisation No-Code basée sur le cloud pour automatiser les flux de travail en ligne

Mais aussi…

Haskell

Haskell est un langage fonctionnel pur connu pour sa fiabilité et sa maintenabilité.

Electron

Electron est un framework permettant de créer des applications multiplateformes en utilisant JavaScript, HTML et CSS.

GraphQL

GraphQL est un langage de requête pour les API qui permet de récupérer uniquement les données nécessaires.

Django

Django est un framework web écrit en Python, conçu pour faciliter et accélérer le développement d’applications web robustes et évolutives.

Laravel

Laravel est un framework PHP qui simplifie le développement web avec ses nombreuses fonctionnalités.

Sylius

Sylius est une plateforme e-commerce flexible basée sur Symfony qui permet de créer des sites e-commerce sur mesure.
